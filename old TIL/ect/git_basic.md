#tips

`git conifg —list` : config의 항목을 볼 수 있다
`git config —global -e` : editor
`git config --global user.name = 'A'`
`git config --global user.email = 'A@gmail.com'`

*core.autocrlf*
윈도우는 true, 맥은 input

*단축키만들기*
`git config —global alias.st status` : status 단축키를 만든다

`git init`
